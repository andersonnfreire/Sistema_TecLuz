# CRUD - Sistema TecLuz

The system was implemented in java, which has the following requirements:

- Insert, list, change operations
  - Client (physical and legal).
  - Vehicle.
  - materials.
  - Service.

Finally, a login for the user to access the system.

# Built With
 - Hibernate
 - Java Swing
 - MySQL
